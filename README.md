# Inventive

**Invent**ory management meets **inventive** engineering.

A portfolio project demonstrating how to build production-grade equipment
booking systems. Features distributed locking, event-driven workflows, state
machines, and pragmatic architecture decisions that balance patterns with
practical engineering.

**Tech Stack:** .NET, PostgreSQL, Redis, RabbitMQ, MassTransit, Hangfire,
Angular, Docker
